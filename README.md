# training-shared

言語非依存の研修資料。

## 構成

| ディレクトリ | 対象 | 格納物 |
|---|---|---|
| `plans/common/` | 教材設計者 / 移植担当 | **言語非依存**のドキュメント。完成形のアプリ仕様（契約）と、研修課題カタログ（狙い・症状・不変条件・難度・移植時の注意。解答情報を含むので受講者には渡さない） |
| `plans/csharp/` | C# 実装担当 | C# リファレンス実装の**固有事項**。固有仕様と課題の仕込み diff 集。契約は `plans/common/` を参照 |
| `db/` | 全員 | **言語非依存**の DB エンティティ表（テーブル定義・関係・制約・列挙・初期データの実値）。移植時の型マッピングの基準 |
| `guides/` | 受講者 | 課題の進め方ガイドと、各課題（A1〜）の詳細（解答なし。`guides/challenges/`）。起動/テストコマンドだけ各言語で読み替える |

新しい言語版バックエンドを起こすときは、カタログをそのまま流用し、各リポジトリ側に「仕込み diff 集」を新規に書く。一部の課題（A1 丸め・A8 例外→HTTP・D1 N+1 など）は言語で罠が変わるため、カタログの「移植時の注意」を必ず参照する。
