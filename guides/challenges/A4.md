# 課題A4（★☆☆）キャンセルしても在庫が戻らない

## 現象

注文をキャンセルするとDB上のステータスは「Cancelled」になるが、在庫が増えない（注文時に減らした在庫が戻らない）。

## 再現手順

1. 商品の在庫数を控えておく（在庫 = N）。
2. その商品を K 個注文する（在庫が N−K に減る）。
3. その注文をキャンセルする。
4. ステータスは「Cancelled」になるが、在庫は N−K のまま。本来は N に戻るはず。

## 期待結果

- キャンセルすると、注文で確保していた在庫が**元に戻る**（N−K → N）。
- ステータスも「Cancelled」になる。
